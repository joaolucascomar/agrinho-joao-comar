Sobre Nós

O “Caminho do Agro” é um projeto educacional do Agrinho que mostra como o produto sai da fazenda e chega à cidade, utilizando de uma aba informativa e interativa. Informações sobre referencias das imagens se encontram no README do projeto

Sobre o Projeto

“Caminho do Agro” é um simulador educativo criado para o Agrinho 2025. Mostra passo a passo o caminho do produto para a cidade cidade.
